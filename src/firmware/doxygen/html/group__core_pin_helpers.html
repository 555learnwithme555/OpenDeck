§ DDR

#define DDR (   x)    (*(&x-1))

Workaround to avoid using DDR and PIN registers. On most AVR models, DDR register is used to define pin direction within port (input or output), and PIN port is used to read state of pin within port. To avoid defining DDR, PORT and PIN register for each pin, these simple macros use the fact that DDR address = PORT address - 1 and PIN address = PORT address - 2. Because of this, only PORT register must be defined, as well as specific pin. Refer to specific AVR model datasheet for more info.
