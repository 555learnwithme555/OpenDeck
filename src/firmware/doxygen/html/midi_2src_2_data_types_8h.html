enum usbMIDIsystemCin_t

Enumeration holding USB-specific values for SysEx messages.

Normally, USB MIDI CIN (cable index number) is just midiMessageType_t shifted left by four bytes, however, System Common/SysEx messages have different values so they're grouped in special enumeration.
